# Release Examples

The packaged release examples are platform-specific. During packaging, this source README is replaced with a target-specific English version that matches the selected release package.

At minimum, each packaged examples directory documents:

- how to run the release validation script,
- how to clean the package-level `build/` directory,
- how to build and run the bundled examples from the package root,
- and, for Windows packages, how to run the bundled SQLite FFI example.
